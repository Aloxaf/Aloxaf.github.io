改进hexo/cafe主题表格显示效果

2018-05-07

文章目录

【注】本文最后更新于 May 7, 2018，文中内容可能已经过时。

总体是根据这篇文章来的 http://ju.outofmemory.cn/entry/130935

进入 themes/cafe/source/css/_partial 编辑article.styl, 将原来.article-entry下的table部分替换成如下代码

  table
    border 1px solid black
    border-collapse collapse
  th
    border-collapse collapse
  td

    padding 1px 8px 1px

编辑hightlight.styl, 在.hight.table下增加border 0px

    table
      margin: 0
      width: auto
      border 0px
      ...

.article-entry
  pre, code
    font-family: "Droid Sans Mono", monospace, font-mono, Consolas, "Source Code Pro", "Microsoft YaHei"

  float: left

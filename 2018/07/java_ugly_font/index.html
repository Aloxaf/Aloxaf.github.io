解决 Arch Linux 下 BurpSuite 打开很丑的问题

2018-07-30 linux

好像很多 Java 程序都有这个毛病.
查了半天 Arch Wiki, 终于找到了: Java Runtime Environment fonts

解决方案: 设置环境变量 _JAVA_OPTIONS='-Dawt.useSystemAAFontSettings=on'
(这么重要的东西怎么还要手动开启??)

直接在 /etc/environment 里加入这句然后重启即可
